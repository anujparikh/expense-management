# Tech 2 Task for Toptal
A small project on expense management for different user roles.
This project uses MEAN stack for development.
## Built with
* Node.js
* Express.js
* Angular.js
* MongoDB
### Prerequisites Installation 
* Node.js
* MongoDB (local)
### Installation
```
    npm install
    node server.js
```
